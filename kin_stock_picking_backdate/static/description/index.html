Stock Picking Back Date

This application helps to manually set date done of Stock Picking.

Basic Features
  • This application provides facility to manually set date done (based on field scheduled date) of Stock Picking (Receipt, Internal Transfer and Delivery Order).
  • The scheduled date you input here will also be used for accounting entry's date if the product is configured with automated stock valuation.
  • No Configuration needed. Just set field Scheduled Date.
